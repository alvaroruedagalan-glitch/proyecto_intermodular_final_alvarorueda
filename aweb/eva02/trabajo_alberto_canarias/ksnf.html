Un centro educativo quiere una página web muy sencilla para recoger algunos datos básicos de un alumno y mostrar distinta información en pantalla según el botón que se pulse. En la página, el usuario deberá poder escribir: • su nombre • su edad • el curso en el que está matriculado La aplicación deberá permitir realizar estas acciones: • Mostrar un mensaje de presentación con el texto exacto: Hola [nombre], tienes [edad] años y estás en [curso]. • Mostrar si el alumno es mayor o menor de edad con uno de estos mensajes exactos: Eres mayor de edad. Eres menor de edad. • Mostrar el nombre en mayúsculas con el texto exacto: Tu nombre en mayúsculas es: [NOMBRE] • Mostrar cuántas letras tiene el nombre con el texto exacto: Tu nombre tiene [número] letras. • Realizar un cambio visual claro en la página al pulsar un botón. Los datos deberán introducirse en la propia página y los resultados deberán aparecer en una zona visible del documento. Document
