Nuevas etiquetas

Vamos a ver dos etiquetas nuevas: <div> y <span>.
La etiqueta <div> sirve para contener otras etiquetas, es decir, es un contenedor.
La etiqueta <span> permite aplicar formato a partes de un texto.
